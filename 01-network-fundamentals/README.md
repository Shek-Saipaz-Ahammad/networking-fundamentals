# 🥇 CCNA 01 - Network Fundamentals

## 📚 Topics Covered

### What is a Network?

A network is a group of connected devices that communicate and share data with each other.

### Computer Network

A computer network is a collection of computers and other devices connected together to exchange data and resources.

### Network vs Internet

* Network: A group of connected devices.
* Internet: A global network of networks.

### Node

A node is any device connected to a network.

**Examples:** PC, Laptop, Server, Router, Switch, Printer.

### Host

A host is a node that has an IP address and can send or receive data on a network.

**Examples:** PC, Laptop, Server, Smartphone.

### Building a Network

Networks are built by connecting devices together using networking devices such as switches, routers, and firewalls.

### Clients

A client is a device that requests services or resources from another device.

**Example:** A laptop opening a website.

### Servers

A server is a device that provides services, resources, or data to clients.

**Example:** A web server hosting a website.

### Servers and Clients

Clients request services, and servers provide those services.

### Switches

A switch connects devices within the same network and forwards data to the correct destination.

### Routers

A router connects different networks and forwards traffic between them.

### Firewalls

A firewall monitors and filters network traffic based on security rules.

## 🎯 Key Takeaway

A computer network consists of hosts, switches, routers, firewalls, clients, and servers working together to enable communication and resource sharing.
