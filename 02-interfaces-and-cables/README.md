# 🌐 02 - Interfaces and Cables

## 🎯 Overview

This topic introduces the basic network interfaces, Ethernet technologies, and cable types used to connect devices in a computer network.

---

## 📚 Topics Covered

### 🌍 Computer Network
A computer network is a group of connected devices that communicate and share resources.

### 🔌 Interfaces
An interface is the physical or logical connection point used to send and receive data.

### 🔗 RJ-45
RJ-45 is the standard connector used for Ethernet cables in most wired networks.

### 🌐 Ethernet
Ethernet is the most common technology used for communication in Local Area Networks (LANs).

### 📡 Network Protocols
Network protocols are rules that define how devices communicate over a network.

### 💾 Bits and Bytes
- **Bit (b):** The smallest unit of data (0 or 1).
- **Byte (B):** A group of 8 bits.

### 📏 Ethernet Standards
Ethernet standards define the speed, cable type, and maximum distance supported by a network connection.

### 🔌 UTP Cable
UTP (Unshielded Twisted Pair) is the most commonly used copper cable for Ethernet networks.

### 🔄 Straight-Through Cable
Used to connect different types of devices, such as a PC to a switch.

### 🔀 Crossover Cable
Used to connect similar devices, such as a switch to another switch, when Auto MDI-X is unavailable.

### ⚙️ Auto MDI-X
Auto MDI-X automatically detects the required cable type, removing the need to choose between straight-through and crossover cables.

### 💡 Fiber-Optic Cable
Fiber-optic cable transmits data using light instead of electrical signals, providing higher speed and longer distances.

### 🟣 Multimode Fiber
Used for shorter distances with lower cost.

### 🟢 Single-Mode Fiber
Used for long-distance communication with higher bandwidth.

### 📖 Fiber-Optic Standards
These standards define the type, speed, and distance supported by fiber-optic connections.

---

## ⚖️ UTP vs Fiber-Optic

| Feature | UTP | Fiber-Optic |
|---------|-----|-------------|
| Signal | Electrical | Light |
| Speed | Lower | Higher |
| Distance | Shorter | Longer |
| Cost | Lower | Higher |

---

## 📌 Key Takeaway

Different interfaces and cable types are used depending on the required speed, distance, cost, and network environment.
