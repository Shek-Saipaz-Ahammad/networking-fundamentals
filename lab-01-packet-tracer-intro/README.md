# Lab 01 - Basic Network Topology

## Objective

Build a basic network topology in Cisco Packet Tracer using routers, switches, firewalls, end devices, and servers.

## Devices Used

* Cisco 2911 Routers (2)
* Cisco 2960 Switches (2)
* Cisco 5505 Firewalls (2)
* PCs (2)
* Servers (2)
* Laptop (Attacker)

## Tasks Performed

* Added all required devices to the workspace.
* Arranged the devices according to the network diagram.
* Connected the devices using the Automatic Connection option in Packet Tracer.
* Verified the physical connectivity between devices.

## Result

Successfully created and connected the network topology in Cisco Packet Tracer.
