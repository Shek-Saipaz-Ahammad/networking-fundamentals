# 🧪 Lab 03 - TCP/IP and OSI Model Analysis

## 🎯 Objective

Analyze network communication using Packet Tracer Simulation Mode and observe how data travels through the TCP/IP and OSI models.

---

## 🛠️ Tools Used

- Cisco Packet Tracer
- Windows Command Prompt (CMD)

---

## 📌 Tasks Performed

- Switched Packet Tracer to **Simulation Mode**.
- Observed network traffic between devices.
- Analyzed the OSI layers involved during communication.
- Used the following commands on **PC1**:
  - `ipconfig`
  - `ipconfig /release`
  - `ipconfig /renew`
- Observed DHCP traffic generated during the IP address renewal process.

---

## 📖 Key Learning

- Learned how devices communicate through different OSI layers.
- Understood the relationship between the TCP/IP and OSI models.
- Observed how DHCP assigns and renews IP addresses.
- Practiced analyzing network traffic using Simulation Mode.
- Used basic Windows networking commands to manage IP configuration.
